# Olá, sou Jeferson Lopes 👋

## Sobre mim

Desenvolvedor Python com 4 anos de experiência no setor de e-commerce, atuando na automação de processos, integração de sistemas e engenharia de dados.
 
Tenho forte experiência no desenvolvimento de soluções para marketplaces como Mercado Livre, Shopee e Amazon, criando integrações robustas via API e otimizando operações como controle de estoque, precificação e gestão de anúncios.
 
Desde o início da operação do setor, participei ativamente de toda a construção do ambiente de e-commerce — desde a criação do primeiro anúncio, a realização da primeira venda até as primeiras análises de dados — o que me proporcionou uma visão completa e estratégica do negócio.
 
Ao longo da minha trajetória:
 
 - Desenvolvi bibliotecas internas para padronizar integrações com APIs 
 - Automatizei processos críticos integrados ao ERP da empresa 
 - Trabalhei com grandes volumes de dados (mais de 16 mil pedidos mensais) 
 - Implementei pipelines de dados utilizando Apache Airflow e Docker 
 
Destaques relevantes:
 - Atuação direta na criação e otimização de anúncios de alta complexidade (ex: produtos com mais de 100 variações e mais de 90 mil anúncios), com controle automatizado via API — contribuindo para alta competitividade nos marketplaces.
 - Contribuição direta para a evolução da empresa, que hoje se posiciona como referência no e-commerce de autopeças, por meio da construção de soluções escaláveis, automações robustas e melhoria contínua dos processos.
 
Atualmente, venho aprofundando meus conhecimentos em Engenharia de Dados, com foco em escalabilidade, confiabilidade e organização de dados, além de estar estudando Java e seu ecossistema, com ênfase em frameworks como Spring Boot e Jakarta EE.
